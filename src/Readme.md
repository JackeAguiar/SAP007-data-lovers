Segundo projeto da quinta geração do bootcamp de front-end da Laboratória. O projeto consiste na criação de uma página web para visualização de um conjunto de dados que se adeque às necessidades do usuário. Dentre os temas sugeridos, escolhemos Pokemon. A partir disso criamos uma pokedex com os dados de 151 pokemons de Kanto da primeira geração.
 Projeto desenvolvido em dupla por [Helena Gonçalves](https://www.github.com/nannayusuf "Github Profile") e [Jackeline Aguiar](https://www.github.com/JackeAguiar "Github Profile")

 Planejamento e Organização
  O planejamento foi inteiramente feito com o método Kanban através da plataforma Notion.

  * Pesquisa
 A pesquisa foi realizada em etapa única com 4 jogadores ativos de Pokemon Go desde seu lançamento com perguntas objetivas visando compreender quais informações seriam úteis na construção da aplicação e definir usabilidade do projeto.

  * História de Usuário 

  A partir da compreensão das necessidades do usuário, elaboramos uma persona com a finalidade de definir o público alvo do projeto.

                         Percebemos com a pesquisa que a maior procura de informações sobre Pokémon é diretamente ligada a consulta de dúvidas sobre os detalhes dos mesmos,

Processo Criativo

O protótipo de baixa fidelidade foi feito inicialmente no canva 


Usamos ele como base para as mudanças futuras de adequação do site durante o processo de desenvolvimento.
